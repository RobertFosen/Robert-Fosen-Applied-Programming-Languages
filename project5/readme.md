 ## What this program does:
 
 This program contains 3 files.  puzzle5.py, which is the python code, puzzle.txt which is
 the puzzle file and wordList.txt which contains all of the words you are potentially looking
 for in the puzzle file.  This programs reads in both files, splits up the puzzle file and
 searches each row and column for every word in the wordList file.
 
  ## How to run the program:
  
  - Download all 3 files and place them in the same directory
  - Open your terminal window and navigate to that directory
  - Type "python ./puzzle5.py"
